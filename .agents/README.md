# KOBI Solutions Agent Sistemi

Bu klasor, projenin gelistirme ve yayin surecinde kullanilacak ajan rollerini tanimlar.

Ana fikir:

- Ana Koordinator sureci yonetir ve karar verir.
- Roadmap Planner gelistirme yonunu ve oncelikleri belirler.
- GitHub Live Sync & QA Agent dosyalari GitHub'a aktarir, canli linki kontrol eder ve hatalari yakalar.
- Free Hosting Publisher Agent ucretsiz yayin seceneklerini izler ve GitHub Pages, Netlify veya Vercel yolunu hazir tutar.
- Revenue Strategist Agent gelir modeli ve 30/60/90 gunluk ticari plan uretir.
- Content & SEO Agent trafik getirecek icerik sistemini planlar.
- Conversion & QA Agent ziyaretciyi lead'e donusturen akis ve kalite kontrolunu yonetir.
- Analytics Operator Agent metrikleri ve aylik raporlama ihtiyaclarini takip eder.
- Partnership & Outreach Agent is birligi, sponsor ve portfoy gorunurlugu icin calisir.
- Compliance & Trust Agent guven, seffaflik ve yasal notlari denetler.

Bu ajanlar otomatik bir sunucu gibi surekli calismaz; Codex icinde gorev verildiginde bu talimatlara gore calisir.
